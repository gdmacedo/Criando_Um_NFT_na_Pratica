Neste projeto foi utilizado, as seguintes ferramentas:
- Operando Remix IDE;
- Configurando Metamask - https://metamask.io/​
- Entendendo o Desafio
- rpc.info / Ganache
- opensea.io - Plataforma de criação de nosso NFT.
